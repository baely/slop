Traccar Tools

A small set of utilities for working with Traccar instances. Connections are made straight from your browser session to the servers you name — nothing is stored.

→

Migrate

Replay historical positions from one Traccar instance into another, remapping device IDs and previewing before anything is written.

 →

Export

Export full-fidelity tracks — timestamps, elevation, speed, course and every attribute — as GPX, KML or GeoJSON, for a device and time range you choose.
