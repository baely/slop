slop design system

The shared design language extracted from ~10 vibe-coded applications. Minimal, functional, and built entirely with vanilla HTML, CSS, and JS.

v1.0 System UI Base-4 Spacing No Frameworks

Principles

Six rules that hold across every project in the repo.

Minimal & functional
Every element earns its place. No decoration for its own sake. If removing it changes nothing, remove it.
Generous whitespace
Let content breathe. Spacing does more for hierarchy than bold type or color ever will.
Clear typographic hierarchy
Size, weight, and color establish importance. Uppercase + letter-spacing for labels. Monospace for data.
Subtle animation
0.15s transitions on interactive elements. Motion confirms interaction without drawing attention to itself.
Flexbox-first layout
Flexbox handles nearly all layout needs. Grid only when the content is truly two-dimensional.
No frameworks
Vanilla HTML, CSS, and JS. No build step, no dependencies, no abstractions between intent and output.

Transitions

Every interactive element gets a transition. The standard is 0.15s — fast enough to feel instant, slow enough to feel smooth.

0.15s — Standard
0.3s — Slow
/* Standard — buttons, inputs, links */
transition: all 0.15s;

/* Slow — panels, modals, large state changes */
transition: all 0.3s;
