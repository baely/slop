slop design system

A shared visual language across slop projects. Flat, minimal, monochromatic. No rounded corners. No dark mode. Just clean grays, a single black accent, and tight type.

Version 1.1 Light only No radius system-ui 13px

Colors

A strictly monochromatic palette. No accent hues except danger red. All text and UI elements live in the black-to-white range with #ddd as the primary border color.

Typography

System font stack at 13px base. Labels are uppercase with letter-spacing. Dates and metadata use Courier New monospace. No web fonts — fast and consistent cross-platform.

Spacing

Compact spacing. Common values: 4, 6, 8, 10, 12, 14, 16, 20, 24, 28, 32, 40px. No CSS custom properties required — use pixel values directly.

No Border Radius

All elements are sharp-cornered. border-radius is never used. Cards, inputs, buttons, modals — all square.

Buttons

All buttons are square (no border-radius). Three semantic variants plus sidebar and navigation helpers. No icon-only buttons — labels are always text.

Primary

Black background, white text. Full-width in forms. Uppercase 12px with tight letter-spacing.

Responsive

Single breakpoint at 640px. On mobile, the sidebar and editor panel are never both visible — the sidebar fills the screen, and the editor overlays it when a note is open. Overlays become bottom sheets. Inputs grow to 16px to prevent iOS auto-zoom.
