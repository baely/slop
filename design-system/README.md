# Design System

Version 1.1 of the shared design system for this monorepo. A minimal, high-contrast component library built on system-ui with #111 on #fff.

## Components

- **Overlay / Auth** — full-screen login card with passphrase input
- **Buttons** — primary, secondary-outline, danger, sidebar
- **Forms** — labelled inputs, error messages
- **App Layout** — sidebar + editor panel split view
- **Note List** — selectable items with title and date
- **Folder Tree** — collapsible folder navigation
- **Mode Toggle** — edit/preview switcher
- **Stats Bar** — word/character counts
- **Modal** — overlay dialog with actions
- **Typography** — headings, body, code, blockquote, lists

## Usage

Link `styles.css` in your app:

```html
<link rel="stylesheet" href="/path/to/styles.css">
```

## Deployment

Deployed as a static site via staticer.
